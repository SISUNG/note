##### 1.常用快捷键总结

| 快捷键   | 功能         | 快捷键       | 功能     |
| -------- | ------------ | ------------ | -------- |
| Alt+F1   | 查找当前文件 | Ctrl+Shift++ | 展开代码 |
| Ctrl+Tab | 切换窗口     | Ctrl+Shift+- | 收缩代码 |
| Ctrl+X   | 剪贴         |              |          |
| Ctrl+B   | 看声明       |              |          |
| Ctrl+N   | 查找类名     |              |          |
| Shift+F9 | 调试         |              |          |
