vscode python Anaconda 配置

1. 安装Anaconda

  • 下载地:https://mirrors.tuna.tsinghua.edu.cn/anaconda/archive/
  • 一路安装下去，注意一点，勾选上 加入环境变量 选项

2. 测试python是否安装成功

  • 打开cmd 输入python,出现下图则说明环境变量没问题.
  • 如果步骤1中没有加入到环境变量.则手动添加环境变量到 系统环境变量的path下。
  • 分别添加: 具体根据自己的安装而定.
D:\Anaconda3
D:\Anaconda3\Scripts
D:\Anaconda3\Library\bin
  • 手动添加环境变量后，最好重启一下计算机，不重启也无妨。再次到cmd下检查python是否安装成功.

3. 安装vscode

  • 新版的Anaconda会提示是否安装vscode，按着操作即可，或者直接去官网下载。
  • 安装完vscode以后，win系统会要求安装git. 下面网址下载git，一路安装下去。
    https://pan.baidu.com/s/1kU5OCOB#list/path=%2Fpub%2Fgit

4. 配置vscode

  • 首先安装一些插件，注意安装完以后，都需要点一下 重新加载
    • python
    • code runner (快捷运行程序)
    • One Dark Pro (不错的主题)
    • vscode-icons (美化文件类型图标)
  • 配置python路径
    • 文件 -> 首选项 -> 设置
    • 更改python路径 搜索python.pythonpath. 如下图所示
    • 把冒号引号中的内容改成自己的python路径. 也可以不改，影响不大，最终我的其他一些设置如下：
{
    "files.autoSave": "afterDelay",//自动保存更改的文件
    "editor.fontSize": 25,//字体
    "python.linting.pylintArgs": [ // 关闭pylint的波浪线提示
        "--disable=W,C,E" 
    ],
    "workbench.colorTheme": "One Dark Pro",
    "workbench.iconTheme": "vscode-icons",
    "explorer.confirmDelete": false,
    "window.zoomLevel": 0,
    "vsicons.dontShowNewVersionMessage": true,
    "python.pythonPath": "python" // 此处可以改成自己的python路径
}

5. 测试

  • 在D盘新建一个文件夹test，在test里新建一个1.py文件
  • 用vscode打开test文件夹
  • 打开1.py 写入如下代码
print('hello world') //python3版本
  • 运行方法
    • 右键 –> run code (使用 code runner 运行程序)
    • ctrl + ~ 调出终端 python 1.py 回车运行文件.
